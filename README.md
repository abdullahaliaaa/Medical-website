## 📖 Project Overview
Yeh ek modern healthcare solution website hai jo patient-centric design par mabni hai. Is project mein **Advanced JavaScript** aur **Typed.js** ka behtareen istemal kiya gaya hai taake user interface interactive aur professional lage. Isme premium animations aur smooth transitions ka use kiya gaya hai jo user experience ko next level par le jati hain.

## ✨ Key Features
* **Interactive Hero Section:** Typed.js ka istemal karke dynamic typing effects create kiye gaye hain.
* **Pro Animations:** Website mein smooth scroll-reveal aur hover animations implement kiye hain.
* **Fully Responsive:** Yeh website har screen size (Desktop, Tablet, Mobile) par perfect chalti hai.
* **Clean & Modern UI:** Medical industry ke standards ke mutabiq ek professional layout aur color palette.
* **Optimized Code:** Clean aur well-structured HTML/CSS jo fast loading mein madad deta hai.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 (Custom Grid & Flexbox)
* **Logic:** Modern JavaScript (ES6+)
* **Libraries:** Typed.js (for animated text)
